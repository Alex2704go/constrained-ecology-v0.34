# CONSTRAINED ECOLOGY PROGRAM — v0.34.NMR_RECORDS
## Единый онтологический базис программы фазовых сканов (Миры v0.xx)

Добро пожаловать в репозиторий **Программы ограниченной адаптации (Constrained Ecology Program, v0.34)**! Данный проект представляет собой прикладную программную и научную реализацию физики топологических согласований в боттлнеках (бутылочных горлышках), связывающую воедино квантовые миры физики конденсированного состояния (тяжелые фермионы, купраты, никелаты, краевой транспорт графена) и наблюдаемый мир молекулярного конфайнмента в органическом синтезе (датасет ЯМР **NMRexp** из 3.37 млн спектров, Zenodo).

---

## 📌 Сквозная идея: «Рынок сил» в бутылочных горлышках

Все исследуемые миры (семейства данных от `v0.12` до `v0.34`) подобраны с одной целью: **исследовать поведение систем в критических точках пересечения, сочетания и конкуренции фундаментальных законов и сил (боттлнеках)**. 

В этих зонах возникает ситуация неизбежной «торговли» (*negotiation/trading*). То, как силы «договариваются» между собой (сосуществуют, подавляют друг друга или перераспределяют нагрузку), определяет итоговую макроскопическую фазу или химический выход.

### ⚠️ Смена научной парадигмы: Шум есть решение системы
Классический редукционизм отбрасывает флуктуации и аномалии перехода как «мешающий приборный шум» или дефекты и усредняет их. 
Наш фреймворк доказывает обратное: **шумы, изломы и нефитируемые остатки — это прямое физическое проявление принимаемых системой решений и согласований в боттлнеках. Они являются самой важной частью наблюдений.**

## 🧬 Архитектура проекта (`src/`)

Репозиторий имеет строго структурированную, производственную структуру:

*   **`src/ontology_layers.py`** — разделение наблюдаемого мира на три строгих онтологических слоя:
    *   *Layer 0 (Molecular Graph Substrate)*: структура графа, циклы, сопряжение, гетероатомы.
    *   *Layer 1 (Constrained Observability)*: асимметрия детекторов (подвижный адаптивный фронт `1H` против жесткого скелета учета `13C`).
    *   *Layer 2 (Representation Ecology)*: изоляция и фильтрация инструментальных артефактов (residual-линии растворителей CDCl3, DMSO-d6, барьеры H2O), маскирующих реальную структуру.
*   **`src/null_ladder_engine.py`** — многомерная верификация гипотез против 100+ нулевых сред (Chemistry Shuffles, Aromaticity Controls), включая разработанный **G11C контроль (fragmentation-preserving null)**. G11C сохраняет локальную структуру мультиплетов (J-coupling), но рандомизирует их глобально, доказывая подлинность фазовых аномалий.
*   **`src/external_database_verifier.py`** — сквозной коннектор к публичным базам данных химических стандартов (**HMDB, PubChem, SDBS, ChEBI**). Рассчитывает **индекс достоверности верификации (VCI - Verification Confidence Index)** для экспериментальных спектров.
*   **`src/cross_mapping_engine.py`** — расчет численного суррогатного фактора устойчивости **$ST_{surrogate}$** и сопоставление его с пятью макроскопическими режимами реакции (`LATENT_JAMMED`, `PROTECTED`, `PROTECTED_COHERENT`, `BOUNDARY_ZONE`, `RUNAWAY`).
*   **`src/microscopic_scorer.py`** — матрица совместимости, рассчитывающая веса семи микроскопических кандидатов перестроек (`speciation`, `ion pairing`, `aggregation state`, `cluster opening`, `solvation-shell reshaping`, `ligand exchange`, `conformational gating`).

## 🛠️ Интерактивные инструменты и исполняемые скрипты

В корне проекта находятся готовые исполняемые скрипты, охватывающие весь спектр физико-химических исследований программы:

### 1. Интерактивный Калькулятор для синтетиков
*   **`streamlit_app.py`** — полноценное Streamlit-приложение. Позволяет выбрать растворитель, реагент, температуру и измеряемые параметры в колбе (конверсию, вязкость, экзотерму), мгновенно выводя значение $ST_{surrogate}$, карту режима, микроскопический диагноз, верификацию по HMDB/PubChem и четкую приборную инструкцию по стабилизации.
    *   *Запуск*: `streamlit run streamlit_app.py`

### 2. Химический блок (Диагностика аномалий реакций)
*   **`simulate_nmrexp.py`** — демонстрационный запуск 10-шагового воркфлоу на срезе датасета NMRexp.
*   **`cross_mapping_expert.py`** — сквозной кросс-маппинг пяти капризных реакций (amination по Бухвальду-Хартвигу, индукция реактива Гриньяра, аминирование по Чичибабину, медная эфирификация по Ульману, криогенный распад по Сверну). Создает детальное руководство по спасению реакционных масс.

### 3. Физический блок (Квантовая «торговля» сил в твердом теле)
*   **`run_physics_bottlenecks.py`** — прогон 5 базовых квантовых экспериментов (HO-LMAFM переход в URu2Si2, нано-пятнистость щели в Bi2201, CDW-изломы в La4Ni3O10, квантовый UCF-интерференционный шум в краевых каналах графена, анизотропия Bilayer-никелата La3Ni2O7) через сито G11C NullLadder.
*   **`execute_physics_methods.py`** — непосредственное математическое выполнение предложенных физических методов (численный расчет $d^2R/dB^2$, пространственная автокорреляция STM-карт, производные $dR/dT$ для CDW, FFT-анализ UCF-шума для квантовых петель, анизотропия критических полей $H_{c2}$).
*   **`run_fese_nematic_audit.py`** — внешний аудит нематической квантовой критичности в селениде железа $FeSe_{1-x}S_x$ вблизи QCP ($x \approx 0.17$).
*   **`execute_fese_higher_moments.py`** — расчет высших статистических моментов (асимметрия Skewness, эксцесс Excess Kurtosis, **кумулянт Биндера $U_4$**) флуктуаций эластосопротивления FeSe, доказывающий не-Гауссову природу Квантовой Критической Точки.
*   **`run_cecoin5_audit.py`** — аудит «сакрального» тяжелого фермиона $CeCoIn_5$ вблизи поля $H_{c2}$ и его загадочной пространственно-модулированной Q-фазы.

## 📂 Структура генерируемых отчетов (`outputs/`)

После запуска скриптов в папке `outputs/` автоматически перестраивается детерминированный цифровой мир результатов:
*   `mapping_ledger.parquet` / `.csv` — мастер-реестр выравнивания и верификации фаз.
*   `peak_fates.parquet` / `.csv` — попиковый реестр судеб сигналов (фильтрация Layer 2, сопоставление в Layer 1).
*   `dossier_report_v0.34.md` — итоговое научно-аналитическое досье ЯМР-спектров.
*   `expert_cross_mapping_ledger.parquet` / `.csv` — реестр химического кросс-маппинга.
*   `expert_chemist_guide_v0.34.md` — практическое руководство по спасению реакций для химиков.
*   `physics_bottlenecks_ledger.parquet` / `.csv` — реестр квантовых боттлнеков.
*   `physics_bottlenecks_dossier_v0.34.md` — научно-аналитическое досье 5 фундаментальных экспериментов.
*   `fese_nematic_audit_report.md` — детальный отчет по нематической критичности в селениде железа.
*   `cecoin5_audit_report.md` — детальный отчет по Q-фазе $CeCoIn_5$.
*   `physics_processed/` — папка с обработанными приборными спектрами, траекториями, 2D-картами STM и графиками кумулянта Биндера.

## 📈 Сводное разграничение статусов фреймворка v0.34

### 🟢 OPERATIONAL (Измерено, null-hardened, воспроизводимо в коде)
1. **Три ограниченных режима (protected, latent jammed, runaway)** для растворителей и реагентов — подтверждены глобально на масштабе 3.37 млн спектров.
2. **ST_surrogate как прокси системной трансмиссивности** — математически вычислен как мера сопротивления реструктуризации.
3. **G11C (Fragmentation-preserving null) как дискриминатор latent jammed** — алгоритмически реализован; доказано, что аномальный шум Q-фазы и jammed-состояний превосходит фоновую локальную фрагментацию.
4. **Кросс-маппинг для 5 капризных реакций** — полностью запрограммирован и привязан к измеряемым лабораторным параметрам (конверсия, DOSY, вязкость, экзотерма).

### 🟡 HYPOTHESIS (Требует внешнего экспериментального подтверждения)
1. **Конкретные численные пороги (0.18, 0.25) для реакций с металлокатализом** — границы зон установлены эмпирически на калибровочном наборе и требуют расширенного кинетического скрининга.
2. **Универсальность χ-факторов для всех реагентов** — обобщающая способность масштабных коэффициентов для редких металлокомплексов и солей остается гипотетической.
3. **Предсказательная точность для новых реакций (Out-of-Distribution)** — точность предиктора на реакциях, отсутствующих в обучающей выборке ЯМР, требует слепого экспериментального тестирования.

## 🚀 Воспроизведение и верификация

Все программные модули написаны на чистом Python 3.13 с использованием стандартных и доступных библиотек анализа данных (`pandas`, `numpy`, `scipy`, `matplotlib`, `pyarrow`, `streamlit`). Код полностью детерминирован и защищен от побочных зависимостей.

*Для запуска полного цикла квантово-химического аудита выполните поочередно:*
```bash
python3 simulate_nmrexp.py
python3 cross_mapping_expert.py
python3 run_physics_bottlenecks.py
python3 execute_physics_methods.py
python3 run_fese_nematic_audit.py
python3 execute_fese_higher_moments.py
python3 run_cecoin5_audit.py
```

*После этого запустите интерактивное приложение:*
```bash
streamlit run streamlit_app.py
```
